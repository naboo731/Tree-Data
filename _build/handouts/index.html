Tree Data Exploration

Introduction

In this exercise you will get familiar with Jupyter notebook and matplotlib.

Setup

First, download the CSV file of the Tree Dataset here:

https://data.sfgov.org/City-Infrastructure/Street-Tree-List/tkzw-k3nq

Make sure to move the file to the directory where you will be working for this exercise.

Next, create a virtual environment and install matplotlib, numpy, and jupyter notebook.

$ virtualenv env
$ source env/bin/activate
$ (env) pip3 install notebook
$ (env) pip3 install matplotlib
$ (env) pip3 install numpy
$ (env) pip3 install pandas

Now, start your Jupyter notebook server.

$ (env) jupyter notebook

This should open your browser to your jupyter notebook tree, but if it does not, open this link: http://localhost:8888/tree.

Lastly, create a new Jupyter Notebook by clicking the dropdown on the upper right that says “New” and then select “Python 3 (pykernal)”.

Step 1: Explore

Conduct a series of exploratory operations on this Tree Dataset using pandas. Write a narrative explanation of how you are exploring the data as you do it using the Markdown functionality of Jupyter notebook.

Things to explore:

  • How many unique zipcodes are represented

  • How many total trees are there

  • What are the various formats for dimensions

  • What are the different Site information options

  • Which is more popular for the caretaker– Private or Department of Public Works?

  • What missing data is there in this dataset? Which columns are impacted? How many rows in each column?

Step 2: Visualize

Using matplotlib and Jupyter notebook, create the following visualizations.

  1. Create a histogram of the number of trees in each zipcode.

  2. Create a histogram of the number of trees cared for by “Private” vs. “DPW”.

Conclusion

When you are finished, push your .ipynb file to your Github to turn in this exercise. As a note, we don’t recommend adding the Tree CSV to Github as it is a very large file.
